Transformers
==========================
Distributions from `flowjax.transformers`. Transformers are simple bijections that can
be parameterised by neural networks, e.g. as in a `CouplingFlow` or `MaskedAutoregressiveFlow`.
This is unrelated to the use of "transformers" in self-attention models.
